# Superpowers artifacts (agent-only)

**Save planning and design output here, not under `docs/`** (GitHub Pages).

| Path | Use |
|------|-----|
| **`plans/`** | All agent planning artifacts: implementation plans from writing-plans (`YYYY-MM-DD-<feature>.md`), design/spec docs from brainstorming (`YYYY-MM-DD-<topic>-design.md`), execution notes a skill asks you to persist. |

This directory is tracked in git so paths in `AGENTS.md` and workflow skills resolve to a real folder.
